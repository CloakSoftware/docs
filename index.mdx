The Cloak Bot SDK is the official TypeScript library for building bots on [Cloak](https://cloak.chat), the end-to-end encrypted chat platform. If you have written a Discord bot, you already know the shape of this API. You create a `Client`, listen for events like `messageCreate`, and call `msg.reply()`.

The difference is what happens underneath. Your bot is a full end-to-end encrypted member of every server it joins. It holds its own identity, encrypts and decrypts messages inside your own process, and the Cloak service only ever relays ciphertext. Nobody in the middle can read what your bot sends or receives.

```ts
import { Client } from '@cloak-software/bot-sdk';

const client = new Client({ token: process.env.CLOAK_TOKEN! });

client.on('ready', () => console.log(`Logged in as ${client.user?.id}`));

client.on('messageCreate', (msg) => {
  if (msg.authorId === client.user?.id) return; // ignore our own messages
  if (msg.content.trim() === '!ping') msg.reply('pong');
});

await client.login();
```

That is a complete, working bot. It answers `!ping` in every channel it can see, across every server it belongs to.

## Requirements

- **Node.js** `20.19+` or `22.12+`. The SDK uses a modern transport that needs a recent Node runtime.
- A **Cloak account** and a **bot token**, both created in the Cloak app.

There is no server for you to run or host. Your bot connects directly to Cloak's hosted service.

<Note>
  A Cloak bot is not wire-compatible with Discord. The API mirrors discord.js so the code feels familiar, but a Cloak bot talks to Cloak, not to Discord.
</Note>
